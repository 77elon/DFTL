# DFTL
This simulator assumes a 3D-NAND.

In fact, I/O are possible, allowing more accurate performance evaluation.

## Params
### Flash Hardware

|Hardware| Default|
|------|---|
|Channel|5|
|Way (Chip)|5|
|Die|5|
|Plane|2|
|Block|32|
|Page|32|

|Block Type| Default|
|------|---|
|Data Block (Writable Block)|10|
|Translation Block|2|
